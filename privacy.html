Política de Privacidade

Última atualização: 19 de março de 2026

O Justo está comprometido em proteger a sua privacidade. Esta Política de Privacidade explica como lidamos com as informações ao usar nosso aplicativo.

1. Coleta e Uso de Informações

O Justo foi projetado para ser um utilitário de produtividade para motoristas de aplicativo. Não coletamos, armazenamos ou transmitimos dados pessoais para servidores externos.

  • Dados de Ofertas de Viagem: O app processa informações de texto (valor, distância e tempo) exibidas na tela de aplicativos de terceiros (como Uber e 99) exclusivamente para calcular a viabilidade financeira da corrida.
  • Processamento Local: Todo o processamento de imagem (OCR) é feito localmente no seu dispositivo através do Google ML Kit. Nenhuma imagem da sua tela é salva ou enviada para nós.

2. Serviço de Acessibilidade (AccessibilityService API)

O Justo utiliza o Serviço de Acessibilidade para:

  • Identificar quando um aplicativo de transporte (Uber/99) está em primeiro plano.
  • Ler os dados das ofertas de viagem exibidas na tela para realizar cálculos automáticos (Ganhos por KM/Minuto).

Privacidade da API: Não utilizamos esta API para monitorar interações privadas, senhas ou dados bancários. O uso é restrito à leitura de ofertas de trabalho enquanto o motorista está em serviço.

3. Permissões Requeridas

  • Serviço de Acessibilidade: Necessário para automação e leitura de tela.
  • Sobreposição de Tela (Display Over Other Apps): Necessário para exibir o botão flutuante com os cálculos sobre os apps de transporte.
  • Armazenamento (Opcional): Apenas para salvar preferências locais de configuração.

4. Segurança de Dados

Como não possuímos servidores de banco de dados para o usuário, não há risco de vazamento de dados de nossa parte. Suas configurações (como o status "Pro") são armazenadas localmente no sistema de preferências do Android.

5. Links para Terceiros

O Justo interage com aplicativos de terceiros (Uber, 99). Não temos controle sobre as políticas de privacidade desses aplicativos e recomendamos que você as leia separadamente.
